Aladdin Persson 님의 코드
- 실제 작동하는 코드보다는 수도코드 정도로 보면 좋을 것 같다.

동빈나 님의 코드

- 나의 경우는 Encoder 파트부터 직접 타이핑해서 구현하기보다 이해하는 식으로 진행하였다.
